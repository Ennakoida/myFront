font-size 속성 : 글씨 크기를 조절할 때 사용

선택자 { font-size : 12px | 2em | 130%; }
    참조사이트
  • 고정크기 px 테스트
  • 가변크기 em 테스트
  • 가변크기 % 테스트

font-weight 속성 : 두꺼운 글씨를 표현할 때 사용

선택자 { font-weight : normal | bold | bolder | lighter | 100 | 400 | 900; }
  • 굵은 글꼴로 변경
  • 원래 굵기보다 더 가늘게
  • 원래 굵기보다 더 굵게

font-style 속성 : 텍스트의 기울임을 지정할 때 사용

선택자 { font-style : normal | italic | oblique; }
  • italic 기울임 글꼴
  • oblique 기울임 글꼴
